# :gear: How to Run
```shell
# Run Server:

$ cd server/
$ yarn install
$ yarn start

# Run Web:

cd web/
$ yarn install
$ yarn start

# Run Mobile:

$ cd mobile/
$ yarn install
$ yarn start
```
